## @dns/utils

A component library for useful utility functions 

### Setup

```shell
yarn
```

### Build

```shell
yarn build
```

### Available utils

```javascript
import {
  camelize,
  camelizeKeys,
  convertMillisToTime,
  dnsLogger,
  forEach,
  generateEmptyArray,
  getSearchParamByName,
  omitKeysFromObject,
  onResizeHandler,
  parseDate,
  parseDateAndTime,
  parseDateAndTimeToFixed,
  standardizeSingleLine,
  stringToColour,
  timeString2ms,
  validateEmail,
  validatePassword,
  validURL,
} from '@dns/utils';
```
