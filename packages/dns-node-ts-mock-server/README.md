## @datns/node-ts-mock-server

This package is for local use only and provides a simple express server that can be enhanced and enriched for multiple @datns services.

### SET-UP

```bash
yarn
```

### BUILD

```bash
yarn build
```

### START

```bash
yarn start
```

### Run in docker

```bash
docker-compose up
```
